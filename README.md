# ORION-64 Block Cipher

ORION-64, kriptografi dersleri kapsamında geliştirilmiş, **eğitsel amaçlı** bir blok şifre algoritmasıdır. 
Algoritma, Substitution–Permutation Network (SPN) yapısını temel alarak tasarlanmıştır.

## Algoritma Özeti
- Şifreleme Türü: Blok Şifre
- Blok Boyutu: 64 bit (8 bayt)
- Anahtar Boyutu: 64 bit (8 bayt)
- Round Sayısı: 3
- Yapı: Substitution–Permutation Network (SPN)

## Proje İçeriği
- `src/` : Python ile yazılmış şifreleme ve deşifreleme kodları
- `docs/` : Proje raporu, sözde kod ve akış diyagramları
- `output/` : Programın çalıştığını gösteren örnek çıktı

## Amaç
Bu projenin amacı, temel kriptografik prensipleri (ikame, permütasyon, anahtar ekleme ve round yapısı) 
uygulamalı olarak göstermek ve geliştirilen algoritmanın analiz edilebilir bir yapıya sahip olmasını sağlamaktır.
